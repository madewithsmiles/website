Project Whisperclose

Full-stack Web Application/Browser Plugin

  • Create an accessible web through voice recognition and navigation.

Description

We believe that accessibility should be a priority for browsers and search engines. With Project Whisper, users can use their voices to access the full power of the internet. We use the Wit.ai API for its Natural Language, Voice Recognition, and Sentiment Analysis capabilities to build a browser that puts accessibility first.
